# Student Attendance App

A simple **React-based Student Attendance Management Application** where users can add students, mark their attendance, and manage the student list.

## Features

* Add new students
* Mark attendance as **Present** or **Absent**
* Toggle student details
* Delete students from the list
* Simple and responsive UI

## Built With

* React
* JavaScript (ES6)
* CSS
* Vite

## How to Run the Project

1. Install dependencies

```
npm install
```

2. Start the development server

```
npm run dev
```

3. Open the browser and go to

```
http://localhost:5173
```

## How the App Works

1. Enter the student name in the input field.
2. Click **Add Student** to add a new student.
3. Click **Details** to show attendance options.
4. Mark the student **Present** or **Absent**.
5. Click **Delete** to remove the student.
